\# 🚀 Automação AWS com Python



Projeto desenvolvido para automatizar criação e gerenciamento de servidores na AWS.



\## 🔧 Tecnologias

\- Python

\- AWS (EC2, S3)

\- Boto3



\## ⚙️ Funcionalidades



✅ Listar instâncias EC2  

✅ Criar servidor automaticamente  

✅ Instalar Apache via script  

✅ Subir página web automaticamente  

✅ Ligar e parar instâncias  



\## 💡 Objetivo



Demonstrar habilidades em automação de infraestrutura e cloud computing.



\## 🚀 Resultado



Servidor criado automaticamente com acesso via IP público exibindo página web.



\---



Projeto desenvolvido para evolução profissional na área de Cloud / DevOps.
